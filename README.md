# Robowars 2.0 – Website Report

## Overview

Robowars 2.0 is a dedicated event website developed to support and promote a robotic combat competition. The website acts as a centralized information platform for participants, organizers, and visitors by presenting event details, rules, schedules, and contact information in a clear and structured manner.

This document serves as the README and technical report for the Robowars 2.0 website, explaining its purpose, features, design approach, and implementation details.

---

## Objectives

* To provide a centralized and reliable source of information for the Robowars 2.0 event
* To ensure smooth navigation and ease of access for users
* To implement a modern, technology-oriented visual design
* To make the website responsive across different devices
* To maintain a clean and scalable code structure

---

## Website Features

* Landing page with event introduction and highlights
* About section describing the Robowars competition
* Rules and regulations for participants
* Event schedule and match flow information
* Team registration guidelines
* Contact and support information
* Interactive UI elements and smooth transitions

---

## UI and UX Design

* Design Theme: Dark and futuristic
* Color Scheme: Dark backgrounds with neon accent highlights
* Typography:

  * Headings use bold, modern sans-serif fonts
  * Body text uses clean and readable fonts for clarity
* Layout Orientation:

  * Grid-based layout structure
  * Section-wise vertical scrolling
  * Clear visual hierarchy for better readability

The overall design reflects innovation, competitiveness, and the technical nature of robotic combat events.

---

## Technology Stack

### Frontend

* HTML5 for structure
* Tailwind CSS for styling
* JavaScript for interactivity

### Styling

* Utility-first CSS approach using Tailwind
* Fully responsive design principles

### Deployment

* The website can be deployed using platforms such as GitHub Pages, Netlify, or Vercel

## How to Run the Project

1. Clone the repository using the following command:

   ```bash
   git clone https://github.com/your-repo/robowars-2.0.git
   ```
2. Open the `index.html` file in a web browser
3. For better performance, a local development server may be used

## Responsiveness

The website is fully responsive and optimized for:

* Mobile devices
* Tablets
* Laptops and desktop systems

Tailwind CSS breakpoints are used to ensure consistent behavior across screen sizes.
